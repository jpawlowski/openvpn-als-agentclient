The Adito Agent

The Adito Agent is a standalone component that is launched on a clients machine via the Adito web interface.

It is responsible for the following :-

  • Setting up and controlling tunnels created in Aditos SSL-Tunnels feature. Both remote and local listeners may be created each supporting multiple tunneled connections.
  • Download and launching application extensions configured using Aditos Network Places feature. This may involve setting up a temporary SSL-Tunnel for use by the application.
  • Setting up and controlling temporary tunnels for use by Tunnel Web Forwards.
It has the following features :-
  • Extendable. Agent extensions may be defined in the extension descriptor and download when the Agent starts up. Extensions may contain a Java class that is invoked at this time. This is currently for example by the Drives extensions that adds drive mapping capabilities to the agent.
  • Debug console may be launched by user to monitor System.out and log output. This is helpful when troubleshooting. It must be enabled by passing the debug option when launching the agent.
  • Port monitor may be launched by user to monitor currently active listeners and the tunnels that are travelling over them.
